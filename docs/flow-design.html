입력 한 줄로 전문가 에이전트 팀이 구성되고, 의존성 순서에 맞춰 자동으로 실행됩니다

하네스 생성
(Harness)
전문가 에이전트 팀과
태스크 의존 그래프를
자동 설계합니다
Backend Dev Frontend Dev QA
→
03
병렬 실행
(Execution)
독립 태스크는 병렬,
의존 태스크는 순차로
실시간 스트리밍 출력
병렬화 WebSocket 승인 게이트
→

(Artifact)

실행 버튼 crontab 알림
